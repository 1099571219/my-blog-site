# 回溯搜索
回溯和递归是相辅相成的，有递归就有回溯
递归函数下面的逻辑就是回溯
回溯法中的递归函数（ backTracking ）一般没有返回值
满足终止条件时，收集结果然后 return
递归函数中的 for 一般用来处理节点
```javascript
function backTracking(参数){
  if('终止条件'){
    //收集结果
    return
  }
  for(/** 集合的元素 */){
    //处理节点
    //递归函数
    //回溯操作
  }
      return
}
```
