# rem布局
**rem 是一个单位**
rem (root em) 是一个相对单位，类似于 em，em 是父元素字体大小
rem 的基准则是相对于 html 元素的字体大小
```typescript
html{font-size:10px}
```
**rem 的优点就是可以通过修改 html 里面的文字大小来改变页面中元素的大小，可以整体的进行控制**
